# Apple Workshop on Human-Centered Machine Learning 2024 - Apple Machine Learning Research

**導入文**

 Apple Machine Learning Researchは、人工知能（AI）技術を基盤とした研究活動を行う組織です。2024年、AppleはHuman-Centered Machine Learning 2024というワークショップを開催し、AI技術を基盤とした機械学習の将来像を掲げています。このワークショップでは、人々の中心を据えた機械学習の可能性とリスクを検討し、将来的な影響を考慮しています。
