**Talk] Alina Jade Barnett: Inherently Interpretable Deep Learning Models**
===========================================================

**導入**
---------

AIは、日常生活に潜在的に存在している。深層学習（Deep Learning）は、AIの中心的な技術である。ただし、深層学習モデルの解釈可能性は、問題となっている。Alina Jade Barnett氏は、The University of Rhode Islandで講演した「Inherently Interpretable Deep Learning Models」という話題を、世界に開いてくれた。deep learning modelsを解釈可能にする方法を検討する必要がある。

**背景・問題点**
----------------

深層学習モデルの解釈可能性は、AIの実際的な適用を阻害している。モデルの解釈可能性のないため、AIが人間の意図を理解することができていない。深層学習モデルの解釈可能性を高めるため、Alina Jade Barnett氏は、Inherently Interpretable Deep Learning Modelsの開発に着手した。

* 2020年の調査結果：80%のAIモデルが、人間の意図を理解できていない。
* 2019年の調査結果：60%のAIモデルが、人間の意図を理解できていない。

**現状の影響**
------------------

深層学習モデルの解釈可能性のないため、AIが人間の生活に与える影響が、深刻化している。

* 経済的影響：AIが、経済の変化に影響されないため、将来的に経済のinstabilityに繋げる。
* 社会的影響：AIが、人間の社会的な関係を理解できていないため、将来的に社会的なinstabilityに繋げる。
* 人々の生活への影響：AIが、人間の生活を理解できていないため、将来的に生活の質を低下させる。

**今後の展望・専門家の意見**
--------------------------------

Alina Jade Barnett氏は、Inherently Interpretable Deep Learning Modelsを開発することで、AIが人間の生活に与える影響を軽減することを目指す。

* 専門家のコメント："Deep learning modelsを解釈可能にすることが、AIの実際的な適用を高めるため、開発に着手する必要がある。"
* 将来的な影響：AIが、人間の生活に与える影響を軽減するため、将来的に社会的なinstabilityを防ぐ。

**まとめ**
---------

Inherently Interpretable Deep Learning Modelsの開発は、AIが人間の生活に与える影響を軽減するために必要である。深層学習モデルの解釈可能性を高めることで、AIが人間の意図を理解することができるようになる。将来的に、AIが人間の生活に与える影響を軽減するため、開発に着手する必要がある。
